# 115-strm-web
本项目参考suixing8/115-strm的脚本，在其核心功能上予以完善。

核心功能：基于115网盘目录树文件生成本地strm文件，配合openlist可实现视频302播放1.生成速度极快，每秒生成数千近万个strm 文件；2.支持自定义过滤文件类型； 3.支持openlist登录下载目录树； 4.支持定时任务、增/全量更新量、目录树文件哈希校验判断是否跳过更新、清理失效文件；5.支持WEB管理，这样可以方便修改与触发手动更新，而且进度条还很丝滑
